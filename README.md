# 🎨 Android App List

A community-style index of Android apps that lean into Material 3 and wallpaper-based dynamic color theming. The goal is simply to make it easy to find apps that already match your system's color palette — it says nothing about how good or useful each app actually is.

There's no single authoritative source for "supports Material You" — developers add it, drop it, or half-implement it all the time — so treat every entry here as a starting point for your own testing, not a guarantee.

## 🏷️ Legend

| Tag | Meaning |
|:-:|---|
| `Free` | Entirely free, no ads, no paywall |
| `Freemium` | Free with optional paid features/upgrades |
| `Paid` | One-time purchase or upfront cost |
| `OSS` | Open Source Software |
| `SPINOFF` | Built on another project's codebase |
| `TWEAK` | Unofficially patched by a third party |
| `💤` | No longer actively maintained |
| `⭐` | Highly recommended / Daily driver |

> A few notes: links point wherever is most convenient (GitHub, F-Droid, Play Store, vendor sites, Telegram). Some apps need an extra toggle buried in settings before dynamic color kicks in. If something's broken or outdated, it happens, these things go stale fast.
